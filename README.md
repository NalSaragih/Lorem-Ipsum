Deskripsi :

pada project ini kami kami berencana untuk membuat sebuah Sosial media System yang dimana mengadaptasi kinerja dari sosial media yang sudah ada, seperti WhatsApp dan Instagram 


Isi project : 
notifikasi - queue
antrian pesan - stack
following - graph
postingan - linked list
like, post -  stack
trending - tree


penjelasan program:
pada program ini nantinya akan ada berbagai sistem yang dipakai 

untuk notifikasi nantinya akan memakai konsep stack/queue dan untuk antrian pesan akan memakai konsep stack, namun seperti yang kita tau kalau untuk membuka pesan itu dibebaskan kepada pengguna sehingga tidak terlalu terpaku kepada stack yang harus dari urutan terakhir dulu atau queue yang harus dari urutan pertama dan untuk sistem following nanti akan menggunakan konsep graph dan kemudian untuk postingan akan menggunakan sistem linked list (double) dan untuk like akan menggunkan konsep stack dikarenakan ada kemungkinan untuk batal saat nge-like sehingga bisa langsung di pop dan terakhir untuk trending akan menggunakan konsep tree prioritas.
