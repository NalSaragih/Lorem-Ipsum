Deskripsi :

pada project ini kami kami berencana untuk membuat sebuah Sosial media System yang dimana mengadaptasi kinerja dari sosial media yang sudah ada, seperti WhatsApp dan Instagram 

penjelasan program:
pada program ini nantinya akan ada sistem notifikasi dan antrian pesan

untuk notifikasi nantinya akan memakai konsep stack/queue dan untuk antrian pesan akan memakai konsep stack, namun seperti yang kita tau kalau untuk membuka pesan itu dibebaskan kepada pengguna sehingga tidak terlalu terpaku kepada stack yang harus dari urutan terakhir dulu atau queue yang harus dari urutan pertama.
